# Google Search Console Submission Guide

## Quick Steps to Submit Your Sitemap to Google

### 1. Access Google Search Console
- Go to: https://search.google.com/search-console
- Sign in with your Google account
- Click "Add Property" 

### 2. Add Your Website
- Select "URL Prefix" option
- Enter: `https://collins-creates.github.io/nutri-check/`
- Click "Continue"

### 3. Verify Ownership
Since this is a GitHub Pages site, use the "HTML file" method:
- Download the HTML verification file from Google
- Add it to your repository root
- Push to GitHub
- Click "Verify" in Google Search Console

### 4. Submit Sitemap
- In Google Search Console, go to "Sitemaps" in the left menu
- Enter: `sitemap.xml`
- Click "Submit"

### 5. Monitor Indexing
- Check "Coverage" report for indexing status
- Monitor "Performance" for search traffic
- Review any errors in "Index Coverage"

## Troubleshooting

### Common Issues:
1. **Sitemap not found**: Ensure sitemap.xml is in repository root
2. **Verification failed**: Check HTML file is accessible
3. **No pages indexed**: Request manual indexing
4. **Low rankings**: Focus on content quality and backlinks

### Solutions:
- Verify sitemap accessibility: `https://collins-creates.github.io/nutri-check/sitemap.xml`
- Check robots.txt allows crawling
- Ensure no crawl errors in Search Console
- Monitor and fix any coverage issues
